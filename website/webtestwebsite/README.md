# WebTest landing page

Static landing page (no framework).

## Run locally

- Open `index.html` in your browser, or use any static file server.

Example:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Customize

- Text/content: `index.html`
- Styles: `styles.css`
- "Check my website" form behavior: `script.js`
